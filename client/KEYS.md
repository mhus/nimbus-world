

# Default Key Bindings

* F5 - Toggle Ego Mode vs. Third Person Mode
* F6 - Toggle Fullscreen
* F8 - Toggle Model Selector Visibility
* F9 - Open Editor Control Widget
* F10 - Open Block Editor Widget
* '.' - Toggle Selection Mode
* '/' - Fire Edit Action (depends on current tool)
* 'W', 'A', 'D', 'S' - Move
* 'Space' - Jump or Interact if in Interaction Mode and Interactive Element is Selected
* 'F' - Toggle Movement Mode (Fly,Walk,Sprint,Crawl)
* 'T' - Toggle Shortcut view visibility and slots
* 'P' - Open Settings/Configuration Main Panel
* 
* '1', '2', '3', '4', '5', '6', '7', '8', '9', '0' - Activate Shortcut Slot
* Mouse - Rotate Camera / Interact with UI Elements
* Mouse Button - Activate Mouse Shortcut Slot
